== Description ==
>Woo Video Gallery  answers your question on how to embed an Youtube or Vimeo video gallery into a woocommerce product description page. 

>It will creates a gallery in the single product page where all videos related to the product are placed.

>It support embedded videos from websites like [youtube](https://support.google.com/youtube/answer/171780?hl=en) and [vimeo](https://vimeo.com/help/faq/sharing-videos/embedding-videos).

> Add the URL of YouTube or Vimeo video in the specific field of the product configuration page and your customers will have the pleasure to watch something dynamic and cutomized.

== Plugin Requirement ==

PHP version : 5.3 and latest
WordPress   : Wordpress 3.0 and latest

== Installation ==

1. Install and activate the [woocommerce plugin](https://wordpress.org/plugins/woocommerce/).
1. Upload the `woo-video-gallery` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create your products through 'the Products' menu option
4. Go to the Edit page of one of the created products. Under the 'Product Data' box, select the 'Video gallery' option at the bottom.
upload the youtube or vimeo video.
5. Update the product.

== Frequently Asked Questions ==

= Can I use embedded videos from websites like youtube or vimeo with the plugin? =
Yes, you can. When you are including videos in your products for gallery, just include embedded videos and paste the URL of the youtube or vimeo page.
